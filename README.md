Student Placement Analysis Dashboard

📌 Project Overview

This project is an interactive Excel dashboard that analyzes student placement outcomes across departments and companies. It identifies placement rates, salary trends, and the relationship between academic performance (CGPA) and placement success — helping visualize which departments and skill areas lead to stronger outcomes.

🗂️ Dataset Description

The dashboard is built on a student placement dataset covering 91 students across 4 departments — CSE, DS, ECE, and IT — with fields including:


Department
CGPA
Placement Status (Placed / Internship / None)
Company
Salary (LPA)


🛠️ Tools & Technologies


Microsoft Excel
Pivot Tables & Pivot Charts
Slicers
KPI Cards


📊 Dashboard Components


KPI Cards — Average Salary, Average CGPA, Max Salary, Placement Rate, Highest CGPA
Department-wise Breakdown — Max salary, average CGPA, and outcome counts (Internship / Placed / None) per department
Company-wise Salary Analysis — Average salary offered by each recruiting company
Outcome Distribution — Overall split of Internship / Placed / None across all students


🎯 Key Insights


Overall placement rate stood at 74%, with an average package of 9.25 LPA and a peak offer of 29.29 LPA.
CSE and DS recorded the highest departmental placement rates (50% each), while IT had the lowest (30%) despite comparable average CGPA across departments.
ECE students had the highest average CGPA (7.68) among all departments.
Product-based and tech companies (Amazon, Google, Microsoft, Flipkart) offered significantly higher average salaries (18–21 LPA) compared to service-based companies (Infosys, TCS, Wipro, Accenture) which averaged 5–6 LPA — highlighting a clear salary gap between company types.
The highest individual CGPA recorded was 9.78.
